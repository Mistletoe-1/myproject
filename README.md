# 课表助手

课表助手用于管理课程安排和作业进度，包含课程表、作业管理、日历视图、数据统计和 Excel 导入功能。默认课程数据为 16 周课表，每周课程时间会轮换，不固定为同一套安排。

## 项目链接

https://coursescheduleassistant.netlify.app/

## 功能概览

- 课程表查看：按第 1-16 周切换查看当周课程。
- 课程管理：支持新增、编辑、删除课程。
- 作业管理：支持新增、编辑、删除作业，标记完成状态，并按状态、日期、课程筛选。
- 日历视图：按日期查看当天截止的作业。
- 数据统计：查看课程数量、作业数量、完成率和未来 7 天作业趋势。
- Excel 导入：支持下载模板并导入课程表。
- 本地保存：课程和作业会保存到当前浏览器。

## 默认课程

默认包含 7 门课程：

- 创新创业
- 信息安全
- 单片机及嵌入式系统
- 计算机网络
- 操作系统
- Python程序设计
- 软件工程

每门课每周 2 节，两节课不会挨着；不同周的上课时间会轮换。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

## Excel 模板格式

模板与课程表页面一致：

- 横向：周一到周日
- 纵向：第 1 节到第 5 节
- 单元格内容按 4 行填写

```text
课程名
教师
上课地点
周次：1-16
```

单周或双周示例：

```text
软件工程
孙老师
教学楼 B-204
周次：1-16 单周
```

## 数据说明

课程和作业数据保存在当前浏览器中。清除浏览器数据会清除已保存内容。

如果需要恢复默认课程和作业，可以点击侧边栏底部的“重置课程与作业”。
