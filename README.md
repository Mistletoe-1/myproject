# 课程表与作业日历

一个基于 React 的课程表与作业管理应用，支持课程表展示、Excel导入、作业管理、日历视图和数据统计等功能。

## 功能特点

### 课程表展示
- 表格形式展示一周课程（周一至周日，1-5节课）
- 每节课显示课程名称、教室、授课老师
- 不同课程用不同颜色区分
- 支持周次切换（1-16周），每周课表不同
- 单周/双周课程自动过滤显示

### 导入课表
- 支持上传 Excel 文件自动解析生成课表
- 支持标准课程表 Excel 模板
- 解析后自动填充到课程表视图中

### 作业管理
- 为每门课添加作业
- 包含作业标题、描述、截止时间、优先级
- 支持标记完成/未完成
- 截止时间临近时红色高亮提醒

### 日历视图
- 以日历形式查看当月所有作业分布
- 点击日期可查看当天要交的作业
- 有作业的日期用圆点标记

### 数据统计
- 统计本学期课程数量、作业总数
- 今日已完成作业数量
- 作业完成状态环形图
- 作业截止趋势折线图

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 5
- **UI 组件**: Ant Design
- **图表库**: Chart.js + react-chartjs-2
- **Excel 解析**: xlsx
- **状态管理**: React Hooks (useState, useEffect)
- **数据持久化**: localStorage

## 使用说明

### 课程表
1. 使用顶部周次下拉列表切换查看不同周次的课表
2. 点击「回到本周」按钮快速跳转到当前周次
3. 点击课程单元格可编辑或删除课程
4. 点击空白单元格可添加新课程

### 作业管理
1. 在作业管理页面点击「添加作业」按钮
2. 选择课程、填写作业信息、设置截止时间和优先级
3. 点击作业卡片可编辑作业信息
4. 点击复选框标记作业完成状态

### 导入课表
1. 进入导入课表页面
2. 点击上传区域选择 Excel 文件
3. 系统自动解析并导入课程数据
4. 导入后可在课程表页面查看

### 数据统计
1. 查看课程数量、作业总数等统计数据
2. 通过图表直观了解作业完成情况和截止趋势
